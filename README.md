 מטרת הכימוס ותוצאתו - 
מטרת הכימוס היא לשמור את הנתונים של האובייקט בתוך המחלקה ולמנוע גישה ישירה אליהם.
העבודה עם הנתונים מתבצעת באמצעות מתודות ו-properties של המחלקה בלבד, וכך נמנעות טעויות.
תוצאת הכימוס היא אובייקט בטוח ומסודר שקל להשתמש בו מבלי לפגוע בשלמות הנתונים שלו.

 מטרת ההורשה ותוצאתה -
מטרת ההורשה היא לאפשר למחלקה אחת להשתמש בתכונות ובמתודות של מחלקת בסיס, ובכך להימנע משכפול קוד.
תוצאת ההורשה היא קוד קצר וברור יותר, המאפשר יצירת מחלקות עם תכונות משותפות לצד התנהגות ייחודית.
